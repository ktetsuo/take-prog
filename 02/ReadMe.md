# プログラミング勉強会第2回

## 事前課題（すべてArduinoを使った課題です）
1. タクトスイッチ1個とLED1個を使い、スイッチを押している間だけLEDが点灯するプログラムを作成してください。
2. 1と同じ回路で、スイッチを押している間だけLEDが0.5秒周期(0.25秒点灯、0.25秒消灯)で点滅するプログラムを作成してください。
3. 1と同じ回路で、スイッチをクリックするたびにLEDの点灯と消灯が入れ替わるプログラムを作成してください。（初期状態は消灯）
4. 1と同じ回路で、スイッチをクリックするたびにLEDが「消灯」と「0.5秒周期の点滅」が入れ替わるプログラムを作成してください。（初期状態は消灯）
5. 1の回路にLEDをもう一つ追加し、スイッチをクリックするたびにLED1の「消灯」と「0.5秒周期の点滅」が入れ替わり、スイッチを長押し（2秒間押し続ける）するたびにLED2の「消灯」と「0.5秒周期の点滅」が入れ替わるプログラムを作成してください。（初期状態は両方消灯）
-- 注意点1: 長押しをしたときは、LED1の状態が変わらないように気をつけてください。
-- 注意点2: 長押しの途中に、LEDの点滅が止まらないように気をつけてください。
